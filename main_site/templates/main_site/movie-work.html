my MDB (Movie Database)

Overview

My MDB is an app that displays full details about movies I have seen since January 2019.

The Problem

I needed a platform to hold the details of the movies I have seen - for no particular reason.
I also needed a project to earn Reat js with, and I thought of doing this with it.

Features

The app fetches just a simple data of movie names aand year of release from my local django admin backend.
It then runs an api query to omdb.com with the details. The full movie details is now displaed in react frontend.

The site is fully responsive.

Tools Used

React (with Redux), Django
